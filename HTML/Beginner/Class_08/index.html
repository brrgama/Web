About links

It is possible to create a veriety of links, like presented in the following examples. It may be useful for contact information.

  • Option: Use the <a> tag with the href attribute and the target _blank to open the link in a new tab.
    Site: Go to this website.

  • Option: Use the <a> tag with the href attribute with the value tel: and a phone number to redirect the user to a call.
    Tel.: Call this number.

  • Option: It is also possible to create a link with whatsapp. For that, the app must be consulted to generate the proper link. The target _blank can be used to open in a new tab.
    Whatssap: Start a conversation with this number using whatsapp.

  • Opetion: The link can refer to a email. The same tags are used, but the value of the href attribute must be mailto:.
    Email: Send an email abc@abc.com
