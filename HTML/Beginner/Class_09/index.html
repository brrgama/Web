About <Lang> attribute

The <Lang> should always be used to declare the language of the element where it is inserted. Usually, as the whole page follows the same language, this attribute is declared inside the <html> tag.
This attribute is meant to assist search engines and browsers.

To know the abbreviations for the languages, refer to ISO 639-1. The following list presents some examples.

  • English: en
  • French> fr
  • Spanish: es
  • Portuguese: pt

One can specify the branch of the language, for example, if it is wanted to specify the US English, one can use the "en-US" value inside the lang attribute.
