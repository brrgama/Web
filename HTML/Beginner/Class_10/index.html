The div tag

The <div> tag is an empty container. This tag is used to define divisions inside the HTML. The layout and the content inside a div is not affected with this tag. The ideia of this tag is to group HTML elements to be styled with CSS and manipulated with scripts (for example, using JavaScript).

The id attribute

The id attribute is used to specify a unique id for an HTML element. It is not possible to have more than one element with the same id in an HTML document.

To refer to an id, it must be used the hash character (#) followed by the id name.
