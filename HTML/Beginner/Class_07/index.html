About lists

Unordered List

The <ul> tag is used to create an unordered list. To create an iten inside this unordered list, the <li> tag must be used. Like presented in Example 01.

Example 01 - List of some birds from the Atlantic Forest in Brazil

  • Saffron Toucanet (Pteroglossus bailloni)
  • Three-toed Jacamars (Jacamaralcyon tridactyla)
  • Dusky-legged Guan (Penelope obscura)
  • Seven-colored Tanager (Tangara fastuosa)

Ordered List

The <ol> tag is used to create an ordered list. To create an item inside the ordered list, the <li> tag must be used. The Example 02 ilustrate this kind of list.

Example 02 - Largest American companies by market capitalization in November/2022

  1. Apple
  2. Microsoft
  3. Google
  4. Amazon

For the ordered list, it can be useful the use of some attributes. A list of some attributes is presented:

  • reversed: reverse the order of the list
  • start: indicates the start position of the list
  • tipy: defines the type of the ordered list (numbers, Roman numerals, letters)

The Example 03 presents the use of those attributes.

Example 03 - Most played instruments

  1. Guitar
  2. Piano
  3. Flute
  4. Drums
