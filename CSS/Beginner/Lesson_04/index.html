About Lengths for Font-Size

The root element (HTML element) have 16 pixels, as can be confirmed with the inspect tool from the browser. It is also possible to change the font-size of any element, including the HTML element.

It is possible to set the font-size of an element to be proportional to the root element with the unit rem, as this very paragraph. Therefore, the text from this paragraph will have 2*16 = 32 pixels.
