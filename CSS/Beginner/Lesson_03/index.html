About Units

There are several units to express length.

There are two types of length units: absolute and relative.

Absolute Lengths

The value used to express the length of an image, for example, will be exactly the defined value.

  • cm: centimeters
  • mm: millimeters
  • in: inches (1 in = 96 px = 2.54 cm)
  • px: pixels (1 px = 1/ 96th of 1 in). But this value may vary with the screen resolution.
  • pt: points (1 pt = 1/72 of 1 in)
  • pc: picas (1 pc = 12 pt)

Relative Lengths

The relative length unit specify a length relative to another length property. It scale better between different rendering medium.

  • em Relative to the font-size of the element (2em means 2 times the size of the current font)
  • ex Relative to the x-height of the current font (rarely used)
  • ch Relative to the width of the "0" (zero)
  • rem Relative to font-size of the root element
  • vw Relative to 1% of the width of the viewport
  • vh Relative to 1% of the height of the viewport
  • vmin Relative to 1% of viewport's smaller dimension
  • vmax Relative to 1% of viewport's larger dimension
  • % Relative to the parent element
