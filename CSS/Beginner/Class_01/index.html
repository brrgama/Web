CSS

The style (including the design, layout and variations for different devices and screen sizes) of a web page is defined with the CSS language. CSS stands for Cascading Style Sheets, and it describes how all HTML elements will appear on the screen. With the CSS one can have control of multiple web pages all at once.

Inline CSS

The most simple way of inserting and using CSS in the HTML code is by the called inline insertion. This method must be used to apply a unique style for a single element. To use inline style, the style attribute must be added to the HTML element. In this attribute, one can change a property of this element by editing its value.

To change the color of a paragraph, for example, inside the p tag, it must be inserted the style attribute with the color property receiving the desired value (in this case, the color), for example, red, like this paragraph.

There are tons of properties that can be edited within the style attribute, another example is presented in this paragraph with the backgroun-color property, that defines the background color of the element content.

Internal CSS

An internal style may be useful if the hole HTML page has a unique style. To define de internal style, the tag <style> must be used inside the head section, like this example. The use of this type of style is also advantagous if it is necessary to change the style of many places in the HTML code, this type facilitates the changing in the whole page. In this mode, the CSS selector must be used to select the HTML element(s) must be styled.

External CSS

Even with the addition of internal CSS in this main page (index.html), secondary pages will not maintain the changes of style defined previously. In the external style sheet, it is possible to change the look of an entire website by changing only one file. It is necessary the use of the <link> element in the head section to include a reference to the external style (style.css). Also in this mode, the CSS selector is used to select the HTML element(s) to be styled.
